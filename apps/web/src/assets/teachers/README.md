# Fotos dos Professores

Este diretório deve conter as fotos dos professores.

## Arquivos necessários:
- `professor-1.jpg` - Foto do Prof. Carlos Silva
- `professor-2.jpg` - Foto da Profa. Ana Costa
- `professor-3.jpg` - Foto do Prof. João Oliveira

## Recomendações:
- Formato: JPG ou PNG
- Proporção: 4:3 ou 1:1 (quadrada)
- Resolução mínima: 600x450px
- Tamanho máximo: 500KB por imagem
- As fotos devem ter boa iluminação e fundo neutro
